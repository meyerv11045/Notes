#### What is Markdown?
- Lightweight markup language w/ plain text formatting syntax
- can be converted to HTML/other formats
- Readable & easy to use

#### What is it used for?
- Readme files (Github)
  - .md file extension
- Forums & blog posts
- Used in many static site generators

#### Things you Can Format:
- Headings
- Lists
- Emphasis
- Links
- Blocks of Code
- Images
- Blockquotes
- Horizontal rules
